利用 return 关键字

利用 return 关键字，可以提前函数返回，避免定义中间变量。

if(condition1) return; if(condition2) return; if(condition3) return ; 将概率大的写在前面，尽可能快地进行 return ，不需要进行后续无谓的匹配 （抛出异常也是同理）
